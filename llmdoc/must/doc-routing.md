## Escalation Hints
- If the question is about why a workflow exists, prefer `overview/` and `architecture/`.
- If the question is about how to execute a recurring maintenance task, prefer `guides/`.
- If the question is about stable filenames, commands, or config surfaces, prefer `reference/`.
